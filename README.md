# lead-automation-assignment# Lead Automation Assignment

## Overview
This project automates collection of leads from a Google Form and processes them using Zapier + OpenAI. 
It then stores cleaned data in Google Sheets, creates a Google Calendar event, sends confirmation email, and displays in a dashboard.

## Steps in Workflow
1. User submits form
2. Zapier triggers
3. OpenAI creates strict JSON summary
4. Data saved to Processed_Leads sheet
5. Google Calendar event created
6. Email sent to user
7. Dashboard updated

## Tools Used
- Zapier (main automation)
- Google Forms
- Google Sheets
- Google Calendar
- OpenAI API
- Gmail
- Looker Studio (dashboard)
